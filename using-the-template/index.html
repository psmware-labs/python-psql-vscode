    A New repository from the Template

    You can create a new repository on your personal account or in the organization from this template. In this example, we are goin to create a new repository from the new template. Whilst you can choose to make the repository either public or private, public repositories are visible to the public, while private repositories are only accessible to you, and people you share them with; I make any work related repositories private.

    1. In the upper-right corner of any page, use the + drop-down menu, and select New repository.

    1. Use the Choose a template drop-down and select the psmware-labs/python-psql-vscode template repository.

    1. Select, domain as the owner, and name the repository, in this example we are using seal as the repo name.

    1. We have added a brief description and then chose Private, if it is not already marks as such.

    1. Click on Create Repository

    1. The repository will start generating.

    1. The repository has been created. And as can be seen, has been created from the template.
